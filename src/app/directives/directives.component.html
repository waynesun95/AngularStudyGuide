This div tag is tied to the directive appExampleDirective found in example.directive.ts

This div will start out highlighted in red. Whenever the mouse enters the div, it will be highlighted in yellow. Whenever the mouse exits the div, it will be highlighted in cyan

It is initially highlighted in red because the constructor sets the background color of the ElementRef instance el to 'red'

It is highlighted in yellow when the mouse enters the div because function associated with @HostListener('mouseenter') sets background color to 'yellow'

It is highlighted in cyan when the mouse exits the div because function associated with @HostListener('mouseexit') sets background color to 'cyan'

NOTE: THIS IS ALL DONE IN example.directive.ts AND NOT IN THE COMPONENT!

What are Directives? Explain the attribute directive

  • Directives are instructions which tell angular to do something
  • Attribute Directives allow you to attach behaviour to elements in the DOM
    • Document Object Model (DOM)
      • An API for valid HTML and well-formed XML documents
      • Defines the logical structure of documents and the way a document is accessed and manipulated
      • Using DOM, yhou can build documents, navigate their structure, add, modify, or delete elements and content
      • Anything found in an HTML/DOM document can be accessed, changed, deleted, or added using the DOM
      • Browser turns HTML tags into elements in a tree using DOM
      • Standard convention for representing and interacting with elements in HTML

How many types of Directives are there in Angular 2?

There are three directives:
  1. Component
  2. Structural
  3. Attribute

How are events attached to a Directive?

  • Need to use the @HostListener decorator. Allows us to subscribe to events of the DOM that host an attribute directive

How do we pass external data into Directives?

  • Passing external data into a directive means we can pass any data from a different component into the directive
  • It can be done using @Input() data binding
